# Task Manager App
Company: Vexocore IT Services Private Limited

This repository contains a minimal Task Manager web app (React) with Firebase integration
and an optional Node/Express backend (for server-side jobs like scheduled notifications).



## Structure
- frontend/  -> React app (open with `npm install` then `npm start`)
- backend/   -> Node/Express (optional server; `npm install` then `npm start`)
- ARCHITECTURE.md
- API_DOCS.md

## Quick start (frontend)
1. cd frontend
2. npm install
3. npm start

## Backend quick start (optional)
1. cd backend
2. npm install
3. set environment variables if needed
4. npm start
